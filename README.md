

Một giao diện cổng thông tin trò chơi (Gaming Portal) đẳng cấp mang phong cách Cyberpunk, được tối ưu hóa cho trải nghiệm người dùng (UX). Đây là dự án bài tập cho môn học **Trải nghiệm người dùng (UX)**.

## 🌟 Tính năng nổi bật

- **Thiết kế Cyberpunk Hiện đại**: Sử dụng bảng màu Neon (Cyan, Purple) tạo cảm giác công nghệ tương lai.
- **Hiệu ứng Quét dòng (Scanlines)**: Mang lại phong cách retro-futuristic độc đáo.
- **Thẻ Game 3D (3D Card Flip)**: Hiệu ứng xoay thẻ mượt mà khi di chuột, hiển thị thông tin chi tiết của trò chơi.
- **Ánh sáng theo con trỏ (Cursor Glow)**: Hiệu ứng ánh sáng neon bám theo chuyển động chuột giúp giao diện sinh động hơn.
- **Tối ưu hóa hiển thị (Responsive)**: Giao diện hoạt động tốt trên cả máy tính và thiết bị di động.
- **Chỉ sử dụng HTML & CSS**: Toàn bộ hiệu ứng được xây dựng thuần túy bằng ngôn ngữ đánh dấu và phong cách trang mà không cần dùng đến thư viện bên thứ ba (ngoại trừ Google Fonts).

## 🛠 Công nghệ sử dụng

- **HTML5**: Cấu trúc ngữ nghĩa, đảm bảo khả năng tiếp cận và SEO.
- **CSS3**: Linear Gradients, Backdrop-filter (Glassmorphism), Keyframe Animations, Flexbox & Grid Layout, 3D Transforms.
- **Google Fonts**: Sử dụng font 'Orbitron' (Tiêu đề) và 'Outfit' (Nội dung) để tạo sự chuyên nghiệp.

## 📂 Cấu trúc thư mục

```text
/UX
├── index.html          # File cấu trúc chính của website
├── style.css           # File định dạng phong cách và hiệu ứng
└── img/                # Thư mục chứa tài nguyên hình ảnh
    ├── gaming_hero_bg.jpg
    ├── game-hanh-dong.jpg
    └── game-dua-xe-15.jpg
```

## 🚀 Cách khởi chạy

1. Clone thư mục dự án về máy.
2. Đảm bảo cấu trúc thư mục hình ảnh (`img/`) đã đầy đủ các file ảnh cần thiết.
3. Mở file `index.html` bằng bất kỳ trình duyệt hiện đại nào (Chrome, Edge, Firefox, Safari).
